# UAEC Invasion Planning Pack

This folder contains the first planning documents for the UAEC Invasion browser game.

Files:

- `PRD.md` — what the game is, what features it needs, and what Version 1 includes
- `architecture.md` — how the code should be organized and how the mechanics should work

Recommended next step:
1. Read `PRD.md`
2. Read `architecture.md`
3. Set up the development tools
4. Create the Vite TypeScript project
5. Start with a placeholder rectangle prototype
